# Google Cloud Professional Cloud Architect Practice Exam

A web-based practice exam application that helps you prepare for the Google Cloud Professional Cloud Architect certification.

## Project Overview

This project works in two steps:
1. Generate the exam PDF and questions using `scrape_and_pdf.py`
2. Run the web UI using `app.py` to practice the questions

## Prerequisites

- Python 3.10 or higher
- pip (Python package installer)
- Web browser (Chrome, Firefox, Safari, or Edge)
- Internet connection (for scraping)
- Docker Desktop (optional, for containerized deployment)
  * [Install Docker Desktop](https://www.docker.com/products/docker-desktop)
  * Ensure Docker Desktop is running before building/running containers

## Option 1: Running Locally

### Step 1: Generate Exam PDF and Questions

1. Create and set up virtual environment:
```bash
# Create project directory
mkdir qa-scraper
cd qa-scraper

# Create virtual environment
python -m venv env

# Activate virtual environment
# Windows:
env\Scripts\activate
# macOS/Linux:
source env/bin/activate
```

2. Install required packages:
```bash
pip install requests==2.31.0
pip install beautifulsoup4==4.12.2
pip install reportlab==4.0.9
```

3. Run the scraper:
```bash
python scrape_and_pdf.py
```

This will:
- Scrape questions from examtopics.com
- Generate professional-cloud-architect-questions.pdf
- Create questions.json for the web UI

### Step 2: Run the Web UI

1. Install Flask (in the same virtual environment):
```bash
pip install flask==3.1.2
```

2. Create project structure:
```bash
mkdir templates static
```

3. Copy application files:
   - app.py → root directory
   - index.html → templates folder
   - style.css, script.js → static folder
   - questions.json → root directory (from Step 1)

4. Start the Flask application:
```bash
python app.py
```

5. Open your web browser and navigate to:
```
http://127.0.0.1:5002
```

## Option 2: Running with Docker

1. Start Docker Desktop:
   - Windows/Mac: Launch Docker Desktop application
   - Linux: Start Docker service
     ```bash
     sudo systemctl start docker
     ```

2. Build the Docker image:
```bash
docker build -t exam-ui .
```

3. Run the container:
```bash
docker run -p 5002:5002 exam-ui
```

4. Access the application:
```
http://localhost:5002
```

## Web UI Features

- Interactive question interface
- Real-time feedback on answers
- Hidden community votes until answer selected
- Pagination with configurable questions per page (1, 5, 10, 50, 100)
- Direct question navigation with smooth scrolling
- Support for multiple correct answers
- Responsive design for all devices

## Troubleshooting

### Step 1: Scraper Issues

1. Connection errors:
   - Check internet connection
   - Verify examtopics.com is accessible
   - Try running scraper again

2. PDF generation fails:
   - Check disk space
   - Ensure write permissions in directory
   - Verify reportlab installation

### Step 2: Web UI Issues

1. Port already in use:
```bash
# Windows
netstat -ano | findstr 5002
taskkill /PID <PID> /F

# macOS/Linux
lsof -i :5002
kill -9 <PID>
```

2. Docker issues:

a. Docker daemon not running:
```bash
# Windows/Mac
Start Docker Desktop application

# Linux
sudo systemctl start docker
```

b. Container/image issues:
```bash
# Remove old containers
docker rm -f $(docker ps -aq)

# Clean up images
docker system prune -a

# Rebuild and run
docker build -t exam-ui .
docker run -p 5002:5002 exam-ui
```

3. File not found errors:
   - Verify all files are in correct directories
   - Check questions.json exists and is readable
   - Ensure templates and static folders exist

## Usage Tips

1. Generate fresh questions:
   - Run scrape_and_pdf.py to get latest questions
   - Wait for both PDF and JSON to be generated
   - Verify files exist before starting web UI

2. Practice efficiently:
   - Start with 5-10 questions per page
   - Answer each question before moving on
   - Use direct navigation for quick review
   - Check community votes after answering
